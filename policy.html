Information Collection and Use

The Application stores most cycle and fertility-related data locally on your device. This includes period start dates, cycle length, ovulation predictions, fertile windows, and user-entered notes. These data are not uploaded to any server unless explicitly stated or required for a particular feature in the future.

No personally identifiable information (such as name, email, phone number, or address) is collected or required to use the Application.

In some cases, the Application may collect non-personal, anonymous technical data, such as:

  • Device type and OS version
  • Crash logs and performance statistics
  • General usage data (e.g., screen views, session duration)

This anonymous data is used solely to improve app performance and user experience.

Third-Party Services
